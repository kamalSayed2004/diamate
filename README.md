## 🛠️ Setup & Run

1. install dependencies

   ```bash
   npm install
   ```

2. run in development

   ```bash
   npm run dev
   ```

# 📔 Important Notes

1. you must modify the config.ts file to match the base apis for diamate docker containers.
2. you must activate an extension to stop cors protection (for ex:Allow CORS: Access-Control-Allow-origin)
